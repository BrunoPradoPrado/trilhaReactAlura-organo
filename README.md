# Organo — Projeto React

## Descrição

Projeto desenvolvido durante a **trilha de estudos da Alura – Explore React com JavaScript**.  
O objetivo é **consolidar os fundamentos do React na prática**, aplicando conceitos essenciais do ecossistema React e do JavaScript moderno.

Este repositório foi criado para **registrar e compartilhar o aprendizado adquirido ao longo da trilha**, servindo também como material de estudo e evolução contínua.

---

## Tecnologias utilizadas

- React
- JavaScript (ES6+)
- CSS
- Create React App
- Git e GitHub

---

## Funcionalidades

- Cadastrar colaboradores
- Remover colaboradores
- Favoritar colaboradores
- Criar novos times
- Alterar a cor dos times
- Organizar colaboradores por time
- Renderização dinâmica de listas e componentes

---

## Status do projeto

🎉✅ **Projeto concluído**  

O projeto Organo foi finalizado como parte da trilha *Explore React com JavaScript* da Alura.  
Novos aprendizados da trilha continuam sendo aplicados em outros repositórios.
